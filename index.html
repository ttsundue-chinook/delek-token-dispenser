🖥️ Desktop USB / System Print Setup 1. Connect your 80mm thermal printer via USB cable
2. Install the printer driver (from manufacturer's CD or website)
3. In Windows: Settings → Printers → Set as default printer
4. In Mac: System Settings → Printers & Scanners → Set default
5. When printing a token, the browser print dialog will open — select your thermal printer
6. Set paper size to 80mm × custom in print preferences for best results
💡 Tip: Skip the print dialog every time In Chrome, go to Settings → Printing → Default printer and set your thermal printer. Then use Ctrl+P → click Print directly without changing settings.

🔌 Web Serial (Chrome Desktop) Setup 1. Use Google Chrome or Edge on Windows/Mac/Linux
2. Connect your 80mm thermal printer via USB cable
3. Click "Connect USB Printer" — browser will list serial/USB ports
4. Select the port your printer is on (e.g. COM3, /dev/ttyUSB0)
5. Tokens will print instantly with no dialog — full ESC/POS control
⚙️ Common baud rates: Most thermal printers use 9600 or 19200 bps. If printing garbled text, adjust below.
⚪ Not Connected
📱 Android Bluetooth Setup 1. Open this app in Chrome for Android
2. Turn your printer ON and set it to pairing/discoverable mode
3. Tap "Connect Printer" — Chrome shows nearby Bluetooth devices
4. Select your printer from the list
5. Tokens will now print wirelessly via Bluetooth
🍎 iPhone/iPad (iOS) Web Bluetooth is not supported on iOS Safari. Switch to System / USB mode and use AirPrint, or connect via your hospital's network printer.
